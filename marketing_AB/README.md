# **Анализ A/B теста маркетингового проекта**

Задача проекта — провести оценку результатов A/B-теста. В распоряжении есть датасет с конверсией покупок.

1. Оценить успешность рекламной компании
2. Если компания успешна, то насколько повлияло количество показанной рекламы на конверсию

Инструменты:

* Рачет статистической значимости различий
* Библиотеки: Pandas, NumPy, Matplotlib, SciPy, statsmodels

Статус проекта: Завершен
